## Technical Details

### Gradients Used
```css
/* Background */
--gradient-1: linear-gradient(135deg, #E0E7FF 0%, #FEF3C7 100%);

/* Card Headers */
--gradient-2: linear-gradient(135deg, #DBEAFE 0%, #E0E7FF 100%);

/* Accents */
--gradient-3: linear-gradient(135deg, #FEF3C7 0%, #FED7AA 100%);
```

### Shadow Softness
```css
--shadow-soft: rgba(0, 0, 0, 0.04); /* Very subtle */
```

### Border Radius
```css
Cards: 20px (very rounded)
Buttons: 12px (smooth)
Inputs: 12px (comfortable)
Progress: 16px (gentle)
```
